Issue Name: Headings and Section

Issue Type: Feature

Issue Description

Problem on using <div> tag
Usage of <div> for defining semantic sections, without defining specific values for the class attributes makes the automation of the outlining algorithm impossible.
Mergin some document
Merging several documents is hard: inclusion of a sub-document in a main document means changing the level of the HTML headings Element so that the outline is kept.
Importance of headings
It is important to not skip one or more heading levels because users determind what they are going to read by looking at headings
